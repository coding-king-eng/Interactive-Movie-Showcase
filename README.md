# Interactive-Movie-Showcase
An interactive front-end movie showcase built with HTML, CSS, and vanilla JS.  Features a dynamic landing page, animated gradient background, flexbox movie gallery with trailer links, and a JavaScript-toggled feedback form. A great practical project for mastering core web development.
# 🎬 Movie Review Hub

A modern and responsive movie review web application built for learning and practicing front-end/web development concepts. This project allows users to explore movies, read reviews, view ratings, and experience a clean, user-friendly interface inspired by popular movie platforms.

---

## 📖 Overview

Movie Review Hub is a personal learning project designed to improve skills in web development, UI/UX design, API integration, and responsive layouts. The application showcases movie information in an engaging way while providing a structured and visually appealing review experience.

Whether you're browsing blockbuster hits or discovering hidden gems, Movie Review Hub offers a simple and intuitive interface for movie enthusiasts.

---

## ✨ Features

* 🎥 Browse and explore movie details
* ⭐ View movie ratings and reviews
* 🔍 Search movies instantly
* 📱 Fully responsive design
* 🎨 Modern and clean user interface
* ⚡ Fast loading experience
* 🌓 User-friendly layout and navigation
* 📊 Dynamic content rendering
* 🔗 API-powered movie data integration
* 💻 Beginner-friendly project structure
